# Z&G AUTO ERP v0.75.0 功能总表

## 本次完成的四项核心升级

### 1. 客户、车辆与车队

- 个人、公司、车队三种客户类型
- 客户电话、备用电话、地址、邮箱、会员等级和备注
- 公司车队、付款条款、信用额度和账单邮箱
- 一个车队可保存多名司机，记录司机姓名、电话、驾照后四位和授权状态
- 车辆保存 VIN、车牌、州、Unit Number、年份、品牌、车型、发动机、里程和常用司机
- 可按客户、电话、公司、司机、VIN、车牌、Unit Number 和工单号搜索
- VIN 自动识别并填入年份、品牌、车型和发动机

### 2. 明细式维修工单

- 客户、车辆、车队、司机、授权联系人和 PO Number
- 客户描述、诊断结果、维修记录和技师
- 多个人工项目：项目、工时、费率、技师、小计
- 多个配件项目：编号、名称、数量、成本、销售价、小计
- 外包、税费、折扣、已付款、欠款和预计毛利润自动计算
- 工单状态：等待检查、等待批准、等待配件、维修中、已完成、已交车、已取消
- 旧版工单人工和配件总额自动兼容导入

### 3. 正式打印模板

- Estimate 报价单
- Repair Order 维修工单
- Invoice 发票
- Receipt 收据
- Letter 纸张布局，正文约 8pt，适合紧凑打印
- 客户打印单不显示配件成本和内部毛利润

### 4. 库存自动联动

- 配件编号、OEM 编号、名称、品牌、供应商、进货价、销售价、库存、最低库存和货架位置
- 采购入库自动增加库存并生成流水
- 工单使用配件自动扣库存
- 修改工单数量时自动补扣或退回差额
- 取消或删除工单自动恢复库存
- 库存不足时阻止保存并提示具体配件
- 低库存和库存成本汇总

## 智能与外部服务

- VIN 自动识别：立即可用
- OCR 车牌识别：手机拍照或选择照片后识别，需联网加载 OCR 组件
- 语音生成维修记录：Edge/Chrome 可用，可复制到工单
- AI 故障诊断：已完成安全服务器接口，配置 OpenAI Key 后启用
- AI 照片分类：已完成安全服务器接口，配置 OpenAI Key 后启用
- 短信通知：已完成 Twilio 服务器接口，配置 Twilio 后启用
- 在线付款：已完成 Stripe Checkout 服务器接口，配置 Stripe 后启用

## 老板首页与财务

- 今日营业额、今日实收和今日毛利润
- 本月营业额、本月实收、本月毛利润、本月支出和本月净经营收益
- 未收款总额
- 维修中、等待批准、等待配件和低库存提醒
- 独立收款记录和支出记录

## 安全与云端

- Supabase 邮箱密码登录
- 第一个账号自动建立 Z&G AUTO REPAIR 组织并成为 owner
- 所有业务数据按组织隔离
- 行级安全策略（RLS）保护数据
- 多设备实时刷新
- AI、短信和付款服务器接口会再次核验登录账号是否属于当前组织
- v0.75.0 升级不删除 v0.74.0 原有业务数据
