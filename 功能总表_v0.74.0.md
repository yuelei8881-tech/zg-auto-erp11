# Z&G AUTO ERP v0.74.0 功能总表

本文件依据完整聊天记录汇总。状态分为：**单机版已实现**、**单机版基础记录**、**云端阶段实现**。

## 1. 老板经营驾驶舱

- 今日营业额、今日支出、今日净额：单机版已实现
- 本月营业额、本月支出、本月净额：单机版已实现
- 开单营业额、实际收款、退款、预计毛利润、经营支出、净经营收益分开统计：单机版已实现
- 未收款、维修中、预约、低库存、待审批、有效保修提醒：单机版已实现
- 本周、本年趋势、技师效率、返修率、客户回头率：云端阶段扩展

## 2. 客户、公司与会员

- 个人、公司、车队三种客户：已实现
- 姓名、电话、会员、积分、储值、备注：已实现
- 会员等级、积分、实际储值和赠送余额分开、套餐次数、到期日：已实现
- 客户历史维修、欠款、车辆、活动和保修关联：数据结构已预留
- 推荐奖励、生日提醒、沉睡客户唤回：提醒中心已预留

## 3. 车队、联系人和司机

- 车队公司、主要联系人、公司电话、信用额度、Net 15/30/45：已实现
- 一个公司多名司机、司机电话、授权维修资格：已实现
- 车辆 Unit Number、公司归属、本次司机和司机电话：已实现
- 送车人与司机不同、维修授权人、PO Number：工单和接车字段已实现
- 月结账单、逾期账龄、车队成本分析：云端阶段扩展

## 4. 车辆与接车证据

- VIN、车牌、年份、品牌、车型、里程：已实现
- 按电话、VIN、车牌、Unit Number、司机、故障码全局搜索：已实现
- 接车里程、油量、送车人、原有损伤：已实现
- 维修前、维修后、交车前故障扫描：已实现
- 外观照片分类、客户确认和签名备注：基础记录已实现
- 手机直接调用相机、照片云存储、车牌 OCR、VIN 自动解码：云端阶段实现

## 5. 预约与维修工单

- 预约日期、时间、客户、车辆、项目、状态：已实现
- 工单号、客户、车辆、司机、PO、客户描述、检查结果、故障码：已实现
- 工时、工时费率、配件销售额、配件成本、税费、折扣、已付款：已实现
- 自动计算人工、总价、欠款和预计毛利润：已实现
- 标准状态：待接车、等待诊断、等待批准、等待配件、维修中、完成、等待付款、已交车、取消
- Estimate、Invoice 和 Repair Order 约 8pt 紧凑打印：已实现基础打印
- 多项工时和多项配件逐行明细、保存工单自动扣库存：下一轮单机增强
- 客户在线批准、电子签名和短信通知：云端阶段实现

## 6. 报价、发票和收款

- 报价单号、金额、批准方式和批准人：已实现
- 发票号、关联工单、金额、已付款、状态：已实现
- 订金、收款、退款、储值支付，支持 Cash/Card/Zelle/Check：已实现
- 已付款修改和退款进入审批：审批记录已实现，真正强制执行需云端
- Receipt、Warranty Repair Order 和月结 Statement 模板：下一轮增强

## 7. 库存、采购和供应商

- 配件编号、OEM、名称、供应商、进价、售价、库存、最低库存、货架：已实现
- 采购入库、工单领用、取消退回、退货、报损、盘点流水：已实现
- 采购单、采购金额、状态和审批人：已实现
- 低库存提醒：已实现
- 工单逐项配件自动扣减/取消退回：下一轮单机增强
- 条码扫码和供应商历史价格分析：云端阶段扩展

## 8. 活动和保修

- 活动名称、日期、优惠内容、保修月数/里程、条款：已实现
- 车辆保修项目、原工单、日期、里程上限、仅配件/仅人工/全部：已实现
- “刹车片一年内配件免费、人工正常收费”可按活动和保修记录配置
- 工单完成自动生成保修、保修回厂自动配件 $0：下一轮增强

## 9. 技师、下属账号和权限

- 技师姓名、专长、工时费率、提成、状态：已实现
- 老板、经理、前台、技师、财务、仓库账号资料和本地 PIN：已实现基础管理
- 每个账号独立模块权限、价格/付款/折扣/删除权限：权限说明已预留
- 邮箱密码登录、多设备会话、老板邀请员工：云端阶段实现

## 10. 多人审批和审计日志

- 修改已付款、优惠、退款、已交车工单、删除工单、库存调整：审批类型已实现
- 1 人或 2 人审批、申请前后值、原因和状态：已实现基础记录
- 所有新增、修改、删除，包括老板操作，写入日志：单机版已实现
- 申请人不能批准自己、两名审批人不能相同、日志不可删除：必须由云端服务器强制执行

## 11. 维修案例、提醒和沟通

- 车型、故障码、症状、原因、维修方法、配件、工时、技师、返修：维修案例库已实现
- 换油、轮胎、刹车、冷却液、变速箱油、注册、保修、会员等提醒：已实现
- 电话、短信、Email、微信和现场沟通记录：已实现
- AI 优先检索本店案例、TSB、维修步骤、工时与配件建议：云端阶段实现

## 12. 数据安全、多设备和外部服务

- JSON 完整导出、验证后恢复：单机版已实现
- 电脑、平板、手机响应式界面：已实现
- 多设备实时同步、云照片、不可篡改日志、真实多账号审批：云端阶段
- Twilio 短信、支付链接、客户门户、AI、OCR、VIN 数据服务：外部接口阶段

## 版本原则

1. v0.74.0 是稳定功能基线，不再混入之前 v0.8/v1.0 的错误代码。
2. 当前先交付可双击查看的单机 HTML，用户确认界面和业务流程后再迁移云端。
3. 云端版必须复制此功能总表逐项验收，不能用占位按钮冒充完成。
